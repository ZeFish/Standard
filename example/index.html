Fine-Art Layout Philosophy

Margin & Standard: Inspired by traditional book design margins
Feature: For emphasized content that needs more space
Bleed: Full-width areas that extend to the edges like print bleeds
This terminology comes from centuries of print design wisdom.

Each class name reflects actual typography and print design terminology, creating a framework that speaks the language of fine-art professionals.

Fine-Art Typography Engine

JavaScript library implementing classical typography rules that CSS cannot handle, inspired by centuries of print design wisdom:

Typography Engine Demo

This paragraph demonstrates "smart quotes", proper em--dashes, ellipsis..., and widow prevention. Notice how two letter words like "of the framework" get non-breaking spaces to prevent awkward line breaks.

French typography: Before punctuation marks like colons : semicolons ; exclamation marks ! and question marks ? we add thin spaces following European conventions.

Typography Rules Applied

Smart Quotes: "straight quotes" → "curly quotes"
Punctuation: -- → em—dash, ... → ellipsis…
Widow Prevention: Non-breaking space before last word
Two-letter Words: "of the" → "of the"
French Spacing: "word :" → "word :"
