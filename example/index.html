Layout Areas

The reading system provides multiple content areas for flexible layout:

  • Content: Default optimal reading width (42rem)
  • Small: Narrower for side notes and asides
  • Accent: Wider for pullquotes and highlights
  • Feature: Even wider for images and media
  • Full: Full container width for backgrounds
